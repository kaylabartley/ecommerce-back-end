# E-commerce Back End Refactored Code 


[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Description 

Refactored code to [https://github.com/coding-boot-camp/fantastic-umbrella](https://github.com/coding-boot-camp/fantastic-umbrella). The challenge was to build the back end for an e-commerce site. I took a working Express.js API and configured it to use Sequelize to interact with a MySQL database.

## Installation

Clone repository from [github.com/kaylabartley/ecommerce-back-end](github.com/kaylabartley/ecommerce-back-end) using git clone.


## Usage 

From your command-line, change your directory to the repository and run application by typing npm start. Then use Insomnia ( [downloadhere](https://insomnia.rest/) )to test the api routes.

[Walkthrough Video](https://youtu.be/IxEZ6ilegqA)
